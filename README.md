# Project name
Chatmobile


# Description
A service that provides webtoons and web novels


# Installation instructions
* Node : 16.x
```bash
yarn dev
```
* build
** Android
```bash
cd <root>/android
./gradlew assembleRelease
```
** IOS
```bash
cd <root>/ios
pod install
Xcode > Product > Archive 
```
